# sphp
Switch php version for ubuntu

## Usage

```terminal
iphp 8.2
sphp 8.2
```

## Alias

Insert to file ~/.bashrc

```bash
alias sphp="path/to/sphp"
alias iphp="path/to/iphp"
```

```terminal
source ~/.bashrc
```

## Aditional

# Ubuntu

```terminal
sudo apt update && apt install -y software-properties-common
sudo add-apt-repository ppa:ondrej/php
```
